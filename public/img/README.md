# Imágenes necesarias para el proyecto

Por favor, agrega las siguientes imágenes en la carpeta `public/img/`:

## Imágenes requeridas:

1. **LogoTupahue.png** - Logo de la iglesia para el navbar
2. **actividades.jpg** - Imagen de fondo para el banner de la página de actividades
3. **inicio.jpg** - Imagen de fondo para el hero de la página principal
4. **oracion.jpeg** - Imagen para la tarjeta de reunión de oración
5. **servicio.jpeg** - Imagen para la tarjeta de servicio dominical
6. **iglesia.jpg** - Imagen de fondo para el banner de ministerios
7. **familia1.jpg** - Imagen de la primera familia pastoral
8. **familia2.jpg** - Imagen de la segunda familia pastoral
9. **familia3.jpg** - Imagen de la tercera familia pastoral
10. **nosotros.jpeg** - Imagen de fondo para el banner de nosotros
11. **favicon.ico** - Ícono del sitio (mover a la carpeta `app/` como `favicon.ico`)

## Ubicación de las imágenes:

- Copia estas imágenes desde tu proyecto Django original
- Ruta en Django: `static/img/`
- Ruta en Next.js: `public/img/`

## Nota:
El favicon.ico debe estar en la carpeta `app/` en lugar de `public/img/` para que Next.js lo reconozca automáticamente.
